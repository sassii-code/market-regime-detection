# Market Regime Detection in Equity Markets

This project analyzes historical NIFTY data to identify market regimes using moving average indicators.

## Objective
To classify market conditions into bullish and bearish regimes based on moving average crossovers.

## Tools
Python  
pandas  
numpy  
matplotlib  
yfinance  

## Method
1. Download historical NIFTY data
2. Calculate moving averages
3. Detect regime changes
4. Visualize bullish and bearish periods

## Result
The project visualizes different market regimes on historical NIFTY price data.
